üìÇ Essential File System Commands

Navigation

cd directory-name
cd .. # Go up one level
cd ~ # Go to home directory
pwd # Print working directory

Master navigation to move efficiently between project folders.

File Operations

ls # List files
ls -la # List with details
mkdir folder-name
touch filename.txt
cp source destination
mv old-name new-name
rm filename

Create, copy, move, and delete files and directories.

File Content

cat filename.txt
head -10 filename.txt
tail -10 filename.txt
grep "search" filename.txt
find . -name "*.js"

View and search file contents efficiently.

Process Management

ps aux # List processes
kill process-id
killall process-name
jobs # Background jobs
nohup command &

Manage running processes and background tasks.

üí° Pro Tips & Shortcuts

Tab Completion

Press Tab to auto-complete file names and commands. Double-tap to see all options.

Command History

Use ‚Üë and ‚Üì arrows to navigate command history. Ctrl+R for reverse search.

Clear Terminal

Ctrl+L or type clear to clean up your terminal.

Stop Commands

Ctrl+C to stop a running command. Ctrl+Z to suspend it.

Multiple Commands

Chain commands with && (if first succeeds) or ; (always run both).

Wildcards

Use * for multiple characters, ? for single character. Example: *.js

Pipes & Redirection

| to pipe output, > to redirect to file, >> to append.

Background Processes

Add & at the end to run commands in background. Use jobs to see them.
